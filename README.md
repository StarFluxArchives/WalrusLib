# WalrusLib

## About

WalrusLib is a library used to assist in the development of Techtonica Mods.

## Usage

Using WalrusLib as a developer, you are required to reference the `WalrusLib.dll` in your C# project.

## Installation

### Requirements

- [BepInEx](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.21) v5.4.21

### Steps

Download the latest version of `WalrusLib.zip` from the [releases tab](https://github.com/StarTechtonicLabs/WalrusLib/releases) and extract it into your `BepInEx/plugins` directory.
